## 커밋 메시지 양식

- [출처, 문제 번호] 문제 제목
  - e.g.
    - `[BOJ 1234] 안녕하세요`
   
## 폴더 구조

- 출처/문제 번호/닉네임/
  - e.g.
    - BOJ/1234/whgdk123/

## PR 규칙

- 닉네임/N주차 형식으로 브랜치 생성
  - e.g. whgdkdyd123-week19
- 모든 팀원들이 리뷰를 마친 후 merge
